User Registration Form (Loaded via AJAX)

This form demonstrates loading configuration from a remote API endpoint using the asyncUrl input.

The form configuration is loaded from /mock-api/user-registration-form.json with automatic caching, retry logic, and error handling.

Features demonstrated:

  • Async form configuration loading with loading spinner
  • 5-minute cache for form configurations (subsequent loads are instant)
  • Automatic retry with exponential backoff on network failures
  • Form validation (required fields, email format, min length)
  • Custom error messages on validators
  • Submit button disabled until form is valid
