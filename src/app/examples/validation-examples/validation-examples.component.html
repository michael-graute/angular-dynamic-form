Validation Examples - All 10 Validator Types

This form demonstrates all available validators with custom error messages and placeholder interpolation.

Validators Demonstrated:
  • required: Username, Email, Country, Checkbox, Tags
  • email: Email field with custom message
  • minLength: Username (min 3 chars) with {{ '{' }}expected{{ '}' }}/{{ '{' }}given{{ '}' }} interpolation
  • maxLength: Username (max 20 chars) with {{ '{' }}expected{{ '}' }}/{{ '{' }}given{{ '}' }} interpolation
  • pattern: Phone number (###-###-####)
  • minNumber: Age (min 18) with {{ '{' }}expected{{ '}' }}/{{ '{' }}given{{ '}' }} interpolation
  • maxNumber: Age (max 30) with {{ '{' }}expected{{ '}' }}/{{ '{' }}given{{ '}' }} interpolation
  • minItems: Tags array (min 2 items) with {{ '{' }}expected{{ '}' }}/{{ '{' }}given{{ '}' }} interpolation
  • maxItems: Tags array (max 5 items) with {{ '{' }}expected{{ '}' }}/{{ '{' }}given{{ '}' }} interpolation
  • inArray: Country selection (only USA, Canada, Mexico allowed)
Testing Tips:
  • Try submitting with empty fields to see "required" errors
  • Enter username with 1-2 chars to see minLength error
  • Enter username with 21+ chars to see maxLength error
  • Enter invalid email to see email validation
  • Enter phone number without dashes to see pattern error
  • Select "Brazil" to see inArray error
  • Enter 15 to Age to see minNumber error
  • Enter 43 to Age to see maxNumber error
  • Add 0-1 tags to see minItems error
  • Add 6+ tags to see maxItems error
  • Try to submit without checking the agreement box
